# OpenWrt NSS

This started out as a private repo for building a variant of the work by [Qosmio](https://github.com/qosmio/openwrt-ipq) refining NSS usability on various Qualcomm platforms.

I personally only have an interest in a single target on the ipq807x platform, and have added a few custom packages as well. If you happen to be building for the Linksys MX4200 v1 and v2, then all you need to do is adjust the R2 output stuff as it'll fail without the correct changes! 😅

Feel free to fork the repo (or just outright copy it, I don't care!) if the work in here is of use to you ❤️
